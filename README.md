# Dodgeball
躲避球（Dodgeball）是一个休闲类的计时小游戏，它使用了JBox2D物理引擎，在游戏中模拟真实的物体碰撞。你需要通过360度摇杆控制自己的蓝球躲避红球的追击，并在躲避过程中尽量吃掉其它颜色的功能球，它们会通过各种方法帮你延长时间。巧妙地操纵360度摇杆，找到规律和合适的加力方法，才能让你的蓝球存活更长的时间！
